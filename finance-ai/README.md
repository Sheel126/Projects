# Finance AI Backend (Spring Boot)

This backend powers a personal finance + AI investment recommendation app.  
It provides secure REST APIs for:

- User accounts
- Income tracking
- Expense tracking
- Savings goals
- Market data ingestion
- AI-driven stock recommendations (via Python microservice)

### Tech Stack
- Java 17+
- Spring Boot 3
- Spring Web + JPA
- PostgreSQL
- Docker
- Python (FastAPI) for AI service

### How to Run
```bash
mvn spring-boot:run
